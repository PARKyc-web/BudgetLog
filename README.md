# BudgetLog

--BudgetLog
** 가계부 서버 및 앱 **

1. 기본적으로 가계부를 지원함
2. 가계부를 작성할 때, 깃허브 처럼 여러개의 Repository를 만들 수 있음
3. Repository에 사람을 초대해서 같이 작성할 수 있음 (ex 가족)
4. 가족은 내가 작성한 내용을 수정할 수 있음. 수정한 기록도 확인 할 수 있음.
5. 그래프 및 원형 차트 등으로 수입 및 지출 확인 기능
6. 수입 및 지출에 대한 선형 그래프, 퍼센트 등 확인 기능
7. 수입 및 지출에 카테고리 설정
8. 카테고리 별 확인 기능
9. Front : Vue, Back : Spring-Boot
10. 여러개의 Repository의 모든 자산을 한번에 확인할 수 잇음
11. Repository에서 특정 인원이 등록한 내용을 확인할 수 있음
